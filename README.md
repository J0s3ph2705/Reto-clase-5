# Números primos: Pseudocódigo y Algoritmo

Datos:
 N: Entero
Instrucciones
 Tomar números > n (x)
  Si x tiene números primos menores (que no sean 1) entonces
   Dividir x en los números primos menores
  Sino entonces
   x es primo
Si algún cociente = 0 entonces
 x no se primo
Sino entonces
 x es primo
